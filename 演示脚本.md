## 演示三：越权攻击拦截

### 步骤 5：低权限 Agent 尝试高权限操作

**操作：** 在主页使用 `agent_basic_demo` 登录，尝试访问 `admin:delete` 资源

**API 等价操作：**
```bash
# 先登录 basic agent
TOKEN=$(curl -s -X POST http://127.0.0.1:8000/api/login \
  -H "Content-Type: application/json" \
  -d '{"agent_id":"agent_basic_demo","api_key":"basic-demo-key"}' | python -c "import sys,json;print(json.load(sys.stdin)['access_token'])")

# 尝试越权操作
curl -X POST http://127.0.0.1:8000/api/delegate/check \
  -H "Content-Type: application/json" \
  -d "{\"token\":\"$TOKEN\",\"action\":\"admin:delete\"}"
```

**预期返回：**
```json
{
  "allowed": false,
  "reason": "Capability admin:delete not in token capabilities",
  "chain": ["agent_basic_demo"],
  "risk_score": 0.8
}
```

## 演示八：四级撤销

### 步骤 16：Token 级撤销

```bash
# 先获取一个 Token
TOKEN=$(curl -s -X POST http://127.0.0.1:8000/api/login \
  -H "Content-Type: application/json" \
  -d '{"agent_id":"doc_agent","api_key":"doc-agent-key"}' | python -c "import sys,json;print(json.load(sys.stdin)['access_token'])")

# 撤销该 Token
curl -X POST http://127.0.0.1:8000/api/delegate/revoke \
  -H "Content-Type: application/json" \
  -d "{\"jti\":\"$(echo $TOKEN | cut -d. -1 | base64 -d 2>/dev/null | python -c 'import sys,json;print(json.load(sys.stdin)[\"jti\"])')\"}"
```

**预期结果：** 该 Token 后续所有请求被拒绝

### 步骤 17：Agent 级撤销

**操作：** 打开 http://127.0.0.1:8000/governance ，点击 external_agent 的「撤销」按钮

**预期结果：**
- external_agent 所有 Token 被撤销
- 状态变为 suspended
- 后续所有请求被拒绝
